# Nil Gohil Portfolio

A premium, world-class, professional monochrome developer portfolio designed to showcase projects, skills, and professional journey. Built with a focus on deep aesthetic engineering, smooth animations, and high performance.

## Features
- **Monochrome Editorial Theme:** Strictly balanced palette of #F9F9F9, white, and pure blacks.
- **Dynamic Projects Showcase:** Interactive 3D hover effects with precise intersection layout.
- **Automated GitHub Integration:** Live-sync API to fetch and render the latest GitHub repositories natively.
- **Smooth Navigation Engineering:** Advanced intersection observers and fluid page scrolling with 0px bounding box intercepts.
- **Responsive Layout:** fully optimized for mobile, tablet, and desktop viewing.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Vanilla Native Styling & Variables)
- Vanilla JavaScript (High Performance DOM manipulation)
- Bootstrap 5.3 (Grid constraints & Utilities)
